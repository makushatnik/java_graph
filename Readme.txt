Drawing
--------------------------------------
Что то типа Paint с использованием Graphics2D.

Есть несколько фигур (объект Shape), которые можно рисовать, сохранить, загрузить в редактор.
Изображения сохраняются в формат XML.

Поддержка Drag&Drop. Правда может еще не работать.

Кнопки и оформление меню возможно еще будут меняться.

Splash-скрин показывается 5 секунд.


Считаю хорошим примером работы по паттерну MVC - есть модели, контроллеры, вьюхи:
   - на каждую фигуру есть модель, и есть свой контроллер.
   - использованы 2 View.


Stars
---------------------------------------
Итак, небольшой проект "Звездное небо" готов.

Скачиваем ShowCosmos.jar, потом берем из папки Stars (исходники проекта) файл start.bat (Для всех Windows),
заменяем предложенную там строку полный путь к скачанному файлу ShowCosmos.jar, создаем ярлык на рабочем столе на файл start.bat.
Все. Можно запускать как заставку, когда уходим от компа.

Количество звезд на экране в каждый момент времени - от 50 до 300.
Звезды существуют от 3 до 8 секунд, потом исчезают, появляются новые.
FrameRate стоит 20 кадров в секунду, но фактически выдерживается около 15.
Сейчас цвета звезд выбираются из 12 предопределенных, если раскомментировать строку, в которой есть вызов
функции getRandomColor() - будет доступ к намного большей палитре цветов.
В ближайшее время небольшая часть алгоритма возможно будет переделана, чтобы увеличить быстродействие.


Возможное дальнейшее развитие проекта:
1. Убрать заголовочную часть приложения. Выход из приложения по комбинации клавиш Ctrl + C.
2. Вынести все настройки во внешний файл - setting.xml.
3. При наведении мышки на планету, показывать ее название.
4. У какой-либо планеты добавить спутник.
